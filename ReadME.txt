Simple TODO web application built with Django.
Users can register, log in, and manage their own tasks.

##
    Live Demo:
    https://todo-app-664n.onrender.com
##

Features:
- User registration and login (authentication)
- Create, complete, and delete tasks
- Tasks are assigned to individual users
- Completed tasks are visually marked (line-through)
- Access protection using login_required

Tech Stack:
- Python 3
- Django
- SQLite (default)
- HTML / CSS
- Git & GitHub

Author
Created as a backend portfolio project.

    git clone https://github.com/yourusername/contact-manager.git
